VPS部署

环境准备
git
nginx

使用 VPS 部署博客的主要思路分为三步：

  • hexo deploy 的时候通过 git 把 public 目录下的博客静态资源推送到远程仓库中
  • 推送更新时触发 Git Hooks 将静态资源克隆到网站根目录下
  • 使用 nginx 作为 Web 服务器提供对博客的 HTTP 访问
